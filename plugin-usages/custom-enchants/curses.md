# 诅咒
红色词条，无法被砂轮去除

### 星鱼来了

| 属性   | 值                                                                                                                                      |
| :----- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| **位置** | 裤子                                                                                                                                    |
| **描述** | 星鱼来了                                                                                                                                |
| **效果** | 装备时头顶会不时冒出爱心粒子（5-10s一次），并且可以持续吸引所有类型的动物（类似对着牛拿着小麦）。 |
