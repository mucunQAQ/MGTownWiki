# 魔咒
蓝色词条，位于主手时可以右键释放魔咒，并进入冷却时间，这段时间内玩家无法使用魔咒。魔咒的冷却时间在物品栏右侧以倒计时表示，冷却完成显示为绿色小球。一件物品只能拥有一个魔咒。在潜行状态下右键不会触发魔咒。

### 剑气

| 属性       | 值                                                                                                                                                                                                                              |
| :--------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **位置**   | 剑                                                                                                                                                                                                                              |
| **等级**   | 1-3                                                                                                                                                                                                                             |
| **描述**   | 劈出一道剑气                                                                                                                                                                                                                    |
| **效果**   | 向十字准心的方向劈出一道横向的剑气（横扫粒子），剑气命中生物时视为玩家用该武器攻击一次该生物。为近战攻击，可以被盾牌格挡。魔咒等级会影响剑气大小。 |
| **冷却时间** | 15s                                                                                                                                                                                                                             |

### 识破

| 属性       | 值                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| :--------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **位置**   | 斧                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **等级**   | 1-3                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **描述**   | 反弹短时间内受到的下一次攻击                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **效果**   | 免疫1s内受到的第一次攻击，并将这次攻击转移给攻击者，若不存在攻击者（如摔落伤害，tnt爆炸，药水），则仅会免疫这次伤害而不反弹。反弹伤害视为攻击转移，会在反弹后结算攻击者的减伤区间。魔咒等级会影响反弹的伤害系数，伤害系数=100%+魔咒等级\*25%，反弹系数为独立乘区。使用反弹成功后会在身边产生粒子效果（如果该武器有黄色词条的话）。使用魔咒后切换物品会提前结束状态。 |
| **冷却时间** | 7s                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
