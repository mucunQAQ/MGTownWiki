# 特别说明

1.  **附魔位置限制:** 附魔的位置用于限制生存玩家从附魔台或铁砧得到的附魔（例如不能给弓附魔锋利），但是可以用指令获得不属于该位置的附魔（例如指令生成一个锋利附魔的弓，近战时锋利附魔可以生效增加伤害）。
2.  **附魔等级限制:** 从附魔台和附魔书获得的附魔等级会有限制（例如锋利最高为锋利5增加3点伤害），但是可以用指令获得更高等级的附魔并且正常使用（例如锋利10可增加5.5点伤害），等级处为非指令情况下能获得的最高等级。
3.  **附魔等级无影响:** 有些附魔的等级无用（例如经验修补、精准采集），标注为“附魔等级无影响”。
4.  **箭矢判定:** 对于箭矢“击中生物或方块”的判定，“穿透”附魔会正常产生多次判定。
5.  **“在水中”判定:** 实体碰撞体积所在的任意一个方块为水或含水方块或装水的炼药锅或气泡柱，或者实体碰撞体积所在的任意一个方块处在露天且下雨的环境，即为实体“在水中”。
6.  **“完整攻击”定义:** “完整攻击”指一次攻击完后，攻击冷却完全后的攻击。
7.  **铁砧词条上限:** 铁砧最多能让一个装备获得7个词条（不计入黄色词条和红色词条），金质装备可以获得9个词条，附魔书没有词条上限。
8.  **铁砧红色词条概率:** 在铁砧内进行合成操作后（重命名不算），不计算红色词条和黄色词条，如果物品已经具有3个及以上的词条，有10%的概率随机获得一个红色词条，如果物品已经具有6个及以上的词条，有50%的概率随机获得一个红色词条，如果物品已经具有9个及以上的词条，有100%的概率随机获得一个红色词条。
9.  **右键功能性物品魔咒:** 对于右键有功能性的物品的魔咒，如手持带有重铸魔咒的锄头对着泥土块右键，会正常生成耕地的同时使用魔咒。
10. **伤害与减伤公式:**
    *   **伤害公式:** `[ ( 基础攻击 + 装备攻击加成 + 药水buff ) * 暴击乘区 (如果暴击，初始倍率为150%) + 附魔 ] * 独立乘区`
    *   **减伤公式:** `伤害 * 护甲值减伤 * 抗性乘区 * 独立乘区`
    *   **乘区规则:** 同区域内存在多种效果为加算（如附魔区的`锋利`附魔和`亡灵杀手`附魔为加算），独立乘区内存在多种效果为乘算。
11. **附魔台词条权重与概率:** 附魔台中有概率获得魔咒、附魔、诅咒。其中权重和概率如下，分为三个档位（权重为2/5/10）。
    *   蓝色词条权重为2
    *   紫色词条权重为2
    *   灰色词条权重为5
    *   黄色词条权重为10
    *   红色词条权重为2
    *   其中会避开冲突词条（如不会同时出现锋利和亡灵杀手），以及黄色词条只会出现一个。
12. **特殊附魔获取途径:** “经验修补”、冰霜行者”、“熔火行者”、“丝袜”、“星鱼”附魔只能通过村民交易、战利品和钓鱼获得，不会出现在附魔台中。“风爆”、“灵魂疾行”、“迅捷潜行”、“勇者”、“重叠网络”、“血吼”、“永恒”、“信念”、“代达罗斯”、“联合”、“愿者上钩”、“万能工具”附魔只能通过战利品和钓鱼获得，不会出现在附魔台、村民交易中。
