# 附魔
紫色词条为宝藏附魔，灰色词条为普通附魔，一件物品只能拥有一个宝藏附魔。

### 枭首

| 属性   | 值                                                                                                                                                                                                                                                             |
| :----- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **词条** | 宝藏附魔                                                                                                                                                                                                                                                       |
| **位置** | 剑/斧                                                                                                                                                                                                                                                          |
| **等级** | 1-3                                                                                                                                                                                                                                                            |
| **描述** | 杀死生物时有概率获得头颅                                                                                                                                                                                                                                       |
| **效果** | 杀死生物时有概率获得头颅。附魔等级会影响获取概率，每一级增加20%获取概率，5级时100%获取。原版已有的头颅（骷髅、凋零骷髅、僵尸、苦力怕、猪灵、末影龙）掉落原版的，没有的掉落对应生物皮肤的玩家头颅。 |
