# Mcmmo

## 技能管理

`/mctop`显示全服等级排名

`/mcrank`显示你的排名

`/mcstats（/stats）`显示你的等级

`/mmopower	（/mmopowerlevel, /powerlevel）WIP)`展示你的实力水平

## 技能详情

`/herbalism`草药术

`/mining`挖矿

`/woodcutting`伐木

`/axes`斧技

`/archery`箭术

## 组队指令

`/party`加入一个组织

`/ptp`传送至组织中的一名成员

`/partychat（/pc,/p`切换队伍聊天或发送队伍聊天消息

## 其他指令

`/mcnotify	（/notify）`打开/关闭mcMMO技能聊天显示通知。

`/mcscoreboard（/mcsb）`管理你的mcMMO积分榜。（已禁用）

`/mcability`切换右键点击是否准备技能。

`/mmocompat`提供有关服务器兼容性和模式的信息。
