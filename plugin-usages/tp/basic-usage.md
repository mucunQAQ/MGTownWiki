# 传送插件

传送相关指令如下：

`/tpa <玩家名> <目标玩家名>`申请传送到指定玩家身边

`/tpahere <玩家名>`申请将指定玩家传送到自己身边

`/back`返回自己上一次死亡或传送前的位置

`/home [家名称]`返回到自己设置的某个家（需提前设置）

`/sethome [家名称]`设置当前位置为某个家（可设置多个）

`/delhome [家名称]`删除指定的家

`/spawn`返回世界出生点或服务器设置的全局出生点

`/warplist`显示公共传送点

`/warp <地点>`传送到指定公共传送点

`/setwarp <名称>`设置公共传送点

`/delwarp <名称>`删除公共传送点
