# 皮肤设置教程

1.前往https://mcskin.littleservice.cn/auth/register 进行注册

 

2.注册完进入https://mcskin.littleservice.cn/user

3.点击左侧角色管理，添加新角色

![Role Management](/imgs/skin/role-management.png)

4.取个名字，点击确定(不用和图里的一样)

 

5.前往皮肤库选择自己喜欢的皮肤

![Skin Bank](/imgs/skin/skin-bank.png)

![Skin Bank List](/imgs/skin/skin-bank-list.png)

6.找到喜欢的皮肤后点击进入材质预览

![Material Details](/imgs/skin/material-details.png)

7.点击立刻使用后选择需要使用皮肤的角色

![Set Skin](/imgs/skin/set-skin.png)

8.在启动器的离线栏内填写创建角色时的名字进入游戏即可显示皮肤

![Demonstration Skin](/imgs/skin/demonstration-skin.png)
